## Being busy
I'm busy a lot these days.

Kids, dogs, work and a bit of self care mean I have so little spare time to do a lot of the things I used to do.

I started using AI coding assistants a few years ago when I was working on a project that required me to write a lot of code quickly. I'd heard about this new tool called Cursor and it was a godsend. I was able to write more code more quickly and more efficiently than I ever had before. It was a great feeling.
